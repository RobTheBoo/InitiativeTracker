📁 Importa dati locali

Indica all'app fino a 3 sorgenti separate sul tuo computer (o telefono): una cartella di immagini, una cartella di stanze, e un file libreria (config.json) con personaggi ed effetti. L'app fa una copia in locale dei dati e da quel momento gioca con la copia: la sorgente non viene mai modificata.

Quando importi di nuovo: i file con stesso nome vengono sovrascritti, quelli con nome diverso si aggiungono a quelli gia' presenti. Le stanze con stesso ID chiedono conferma.

🎯 Tre sorgenti separate

📋 Come deve essere strutturata la sorgente (clicca) 
Cartella IMMAGINI (con sub-cartelle)
<cartella>/
├── heroes/    *.png *.jpg *.webp ...
├── enemies/
├── allies/
└── summons/

Cartella STANZE
<cartella>/
├── <id-stanza-1>.json   (esportato dall'app)
├── <id-stanza-2>.json
└── ...

File LIBRERIA (singolo)
config.json   (struttura: heroes, enemies, allies, summons, effects)
Categoria:

I puntamenti vengono ricordati. L'app salva tutti i dati importati nella propria cartella di lavoro (app-data): i file della sorgente non vengono mai modificati. Su Android l'app usa il file picker di sistema. I file selezionati vengono copiati nella memoria interna dell'app (app-data) e da li' usati per giocare. La sorgente non viene mai modificata.

⚙️ Modalita' classica: cartella unica (sincronizza un'unica cartella OneDrive/Drive)

Punta l'app a un'unica cartella locale (anche dentro OneDrive/Drive sincronizzato dall'OS). L'app legge/scrive li' usando una struttura standard.

📁 Popola cartella di lavoro

Scegli una cartella (parte dalla cartella dati del programma, dove sono già le immagini) e l'app la inizializza con tutte le sottocartelle giuste (images/heroes, images/enemies, images/allies, images/summons, rooms). Da li' puoi droppare a mano nuove immagini. Se la cartella la metti in OneDrive/Drive/Dropbox i dati si sincronizzano automaticamente su tutti i tuoi PC.

Nessuna cartella configurata.

🔄 Importa / Esporta

Per le stanze gia' esistenti localmente, l'import chiedera' conferma prima di sovrascrivere.

📋 Struttura della cartella (clicca per dettagli) 
<cartella>/
├── manifest.json        Metadati (versione, timestamp)
├── README.md            Auto-generato dall'app
├── config.json          Libreria personaggi (eroi, nemici, alleati, evocazioni, effetti)
├── images/
│   ├── heroes/          Avatar eroi (PNG/JPG/WebP/GIF/SVG)
│   ├── enemies/         Avatar nemici
│   ├── allies/          Avatar PNG / NPC
│   └── summons/         Avatar evocazioni
└── rooms/
    └── <room-id>.json   Una stanza per file (metadati + gameState)

La prima volta che esporti, l'app crea tutto. Da quel momento puoi tenere la cartella in OneDrive e accedere agli stessi dati da casa, ufficio, portatile.
