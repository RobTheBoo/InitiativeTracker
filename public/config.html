☁️ Backup immagini in cloud

Carica le immagini di eroi, nemici, alleati ed evocazioni su OneDrive e/o Google Drive. Cosi' puoi cambiare PC, formattare, riavviare l'app e ritrovare tutte le immagini.
Il database SQLite resta sempre locale (su cloud si corromperebbe). Solo le immagini vanno in cloud.
Puoi connettere entrambi i provider: l'upload va in automatico su tutti quelli connessi.

☁️ OneDrive Personale

Caricamento…
⚙️ Setup Azure (una-tantum, gratis, 1 minuto)
  1. Vai su portal.azure.com (login col tuo Microsoft account)
  2. Cerca App registrations → + New registration
  3. Nome: RPG Tracker · Account types: Personal Microsoft accounts only · Redirect URI vuoto · Register
  4. Authentication → in fondo "Allow public client flows" = Sì → Save
  5. Overview → copia Application (client) ID

📂 Google Drive

Caricamento…
⚙️ Setup Google Cloud (una-tantum, gratis, 2 minuti)
  1. Vai su console.cloud.google.com (login col tuo Google account)
  2. Crea un nuovo progetto (es. "RPG Tracker") → Select project
  3. APIs & Services → Library → cerca Google Drive API → Enable
  4. APIs & Services → OAuth consent screen → User Type External → compila i campi obbligatori → in Test users aggiungi il tuo email
  5. APIs & Services → Credentials → + Create Credentials → OAuth client ID
  6. Application type: Desktop app · Name: RPG Tracker → Create
  7. Copia il Client ID (NON serve il client secret)

Lo scope usato e' drive.file: l'app vede SOLO i file che ha creato lei, non tutto il tuo Drive.
