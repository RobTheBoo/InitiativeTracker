⚙️ Setup iniziale (una-tantum) — Registra l'app Azure

Microsoft richiede di registrare un'"app" nel tuo account Azure (gratis, 1 minuto). E' come dare un nome al permesso che concedi: serve solo a tracciare chi puo' accedere ai tuoi file.

  1. Vai su portal.azure.com (login con il tuo Microsoft account)
  2. Cerca "App registrations" in alto → clic + New registration
  3. Nome: RPG Tracker (o quello che vuoi)
  4. Account types: "Personal Microsoft accounts only"
  5. Redirect URI: lascia vuoto, clicca Register
  6. Vai a Authentication → in fondo: "Allow public client flows" = Sì → Save
  7. Torna su Overview, copia il valore di "Application (client) ID"
